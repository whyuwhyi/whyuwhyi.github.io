Struct Update Syntax

可以利用已有结构体创建新的结构体。如：

struct Point {
    x: i32,
    y: i32,
    z: i31,
}

let p1 = Point { x: 0, y: 0, z: 0 };
let p2 = Point { x: 1, ..p1 };
